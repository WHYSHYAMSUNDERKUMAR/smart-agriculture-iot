# 🌱 Smart Agriculture Monitoring System (IoT + Web)

## 📌 Description
This project is an IoT-based smart agriculture monitoring system designed to monitor environmental conditions like soil moisture, temperature, and humidity in real-time using ESP32 and sensors. The system provides automated irrigation control and a web-based dashboard for remote monitoring and decision-making.

## 🚀 Key Features
- Real-time monitoring of temperature, humidity, and soil moisture
- Automatic irrigation system based on soil moisture levels
- Web-based dashboard for remote monitoring
- Data visualization using charts and graphs
- Alert system for extreme conditions (low moisture, high temperature)
- Historical data analysis for better decision-making

## 🛠️ Technologies Used
- ESP32 Microcontroller
- DHT11 Sensor
- Soil Moisture Sensor
- PHP, HTML, CSS, JavaScript
- MySQL Database
- Arduino IDE
- XAMPP Server

## ⚙️ How It Works
- Sensors collect real-time environmental data
- ESP32 processes data and sends it to the web server using HTTP
- Data is stored in MySQL database
- Web dashboard displays real-time data and analysis
- System automatically controls irrigation and cooling based on thresholds

## 📊 System Components
- ESP32 (Main Controller)
- Soil Moisture Sensor
- DHT11 (Temperature & Humidity)
- Water Pump (Auto Irrigation)
- Cooling Fan
- I2C Display
- Buzzer (Alerts)

## 📈 Results
- Improved water usage efficiency
- Reduced manual monitoring
- Real-time remote access for farmers
- Better decision-making using data analysis
